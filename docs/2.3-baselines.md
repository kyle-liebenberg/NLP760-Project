# 2.3 Classical baselines (`src/baselines.py`)

This script trains and evaluates **five traditional NLP baselines** on the isiZulu authorship dataset. The results set the bar the CNN-LSTM subword model must beat to justify the added complexity.

## Purpose

Before investing in a neural architecture, we need reproducible benchmarks on the **same train/test split** as the rest of the project. These models use bag-of-words and TF-IDF features with linear classifiers, fast to train, interpretable, and strong on small text-classification tasks.

Character n-gram models are included because isiZulu morphology is often visible at the subword level (shared roots across inflected forms), which word-level bag-of-words can miss.

## Prerequisites

Run [`src/preprocessing.py`](2.1-preprocessing.md) first:

- `data/splits/train.csv`
- `data/splits/test.csv`

The validation split is loaded for logging but **not used** for training or model selection; evaluation is on the held-out test set only.

Baselines operate on **chunked** text (the same augmented splits used by the neural pipeline), not on full raw articles.

## Models

All pipelines use `sklearn.pipeline.Pipeline` (vectoriser → classifier).

| Pipeline name | Features | Classifier |
|---------------|----------|------------|
| `BoW_word_SVM` | Word unigrams + bigrams (`CountVectorizer`, `ngram_range=(1,2)`) | `LinearSVC` |
| `BoW_char_SVM` | Character n-grams 2–4 (`char_wb`) | `LinearSVC` |
| `TFIDF_word_SVM` | Word TF-IDF (1–2 grams), `sublinear_tf=True` | `LinearSVC` |
| `TFIDF_char_SVM` | Character TF-IDF (2–4 grams), `sublinear_tf=True` | `LinearSVC` |
| `TFIDF_word_LogReg` | Word TF-IDF (1–2 grams) | `LogisticRegression` (lbfgs) |

Shared settings:

| Parameter | Value |
|-----------|-------|
| `MAX_FEATURES` | 10,000 |
| `RANDOM_STATE` | 42 |
| `LinearSVC` | `C=1.0`, `max_iter=2000` |
| `LogisticRegression` | `C=1.0`, `max_iter=1000`, `solver="lbfgs"` |

`char_wb` analyser uses character n-grams inside word boundaries, which tends to work better than raw character sequences for morphologically rich text.

## Evaluation

- **Primary metric:** macro-averaged F1; treats all 11 authors equally despite mild class imbalance.
- **Also reported:** macro precision, macro recall, accuracy, per-class classification report.
- **Test set:** 306 chunks from `test.csv` (author names as string labels).
- **Train set:** 1,428 chunks from `train.csv`.

## Main functions

| Function | Role |
|----------|------|
| `load_splits` | Loads train and test CSVs (val loaded for logging only) |
| `build_pipelines` | Constructs the five sklearn pipelines |
| `evaluate_all` | Fits each pipeline on train, predicts on test, collects metrics |
| `print_summary_table` | Prints comparison table and names the best model |
| `plot_comparison` | Saves `baseline_comparison.png` |
| `plot_confusion_matrix` | Heatmap for the best model (normalised by true class) |
| `save_results` | Writes JSON report and metadata |
| `run` | Orchestrates training, evaluation, and figure export |

## Outputs

| File | Description |
|------|-------------|
| `outputs/metrics/baseline_report.json` | Full metrics and per-class reports for all five models |
| `outputs/metrics/baseline_metadata.json` | `random_state`, `max_features`, dataset sizes, model names |
| `outputs/figures/baseline_comparison.png` | Bar chart of F1 macro vs. accuracy per model |
| `outputs/figures/confusion_matrix_<best_model>.png` | Normalised confusion matrix for the best model |

## Latest results (last run)

Evaluated on **306 test chunks**; trained on **1,428 train chunks**.

| Model | F1 (macro) | Precision (macro) | Recall (macro) | Accuracy |
|-------|------------|-------------------|----------------|----------|
| BoW_word_SVM | 0.9034 | 0.9064 | 0.9025 | 0.9020 |
| BoW_char_SVM | 0.8776 | 0.8812 | 0.8788 | 0.8758 |
| **TFIDF_word_SVM** | **0.9360** | **0.9387** | **0.9360** | **0.9359** |
| TFIDF_char_SVM | 0.9149 | 0.9175 | 0.9157 | 0.9150 |
| TFIDF_word_LogReg | 0.8971 | 0.9002 | 0.8985 | 0.8954 |

**Best baseline:** `TFIDF_word_SVM` with **F1 macro = 0.9360**.

The CNN-LSTM should exceed **F1 = 0.9360** on the same test split to demonstrate a clear gain over this benchmark. The latest deep-learning run (`dl_report.json`) reports **F1 macro = 0.8740** - below the best baseline on the augmented chunks.

### Interpretation

- TF-IDF and character models substantially outperform plain word BoW, supporting the hypothesis that subword-level patterns matter for isiZulu.
- On chunked data, **word-level TF-IDF + linear SVM** is the strongest classical model.
- Macro F1 still reflects harder authors; inspect per-class entries in `baseline_report.json` for detail.

## Usage

From the project root:

**macOS / Linux:**

```bash
python3 src/baselines.py
```

**Windows:**

```cmd
python src\baselines.py
```

Console output includes a summary table and paths to saved figures and JSON. This step is **phase 3** of [`run_pipeline.py`](../run_pipeline.py).

## Relationship to the tokeniser

| Component | Input | Feature space |
|-----------|-------|----------------|
| `baselines.py` | Raw `text` from CSV chunks | sklearn n-grams |
| `tokenizer.py` | Raw `text` → BPE IDs | Learned subword vocabulary for neural model |

Baselines do **not** use `X_train.npy` or the BPE tokeniser; they are an independent reference line on the same splits.
