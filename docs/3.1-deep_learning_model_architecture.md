## 3.1. Deep Learning Model Architecture (`src/cnn_lstm_model.py`)

### Overview

A hybrid, dual-track Functional Neural Network engineered using TensorFlow/Keras. The network integrates spatial convolutional feature extractors with an LSTM sequential track and a global keyword skip-connection to achieve balanced, stable authorship attribution on low-resource text distributions.

### Important Technical Details

* **Functional API Design:** The model abandons the standard Keras Sequential pipeline in favor of a functional graph layout to support non-linear bypass routing (skip connections).
* **Trainable Footprint:** 521,099 Total Parameters (520,715 Trainable). This represents an optimal capacity footprint for intermediate text corpora.

### What it Does

It ingests tokenized, padded sequences of BPE subword IDs and transforms them into continuous, high-dimensional probability spaces, ultimately outputting a SoftMax distribution across the 11 target isiZulu journalists.

### Why it Does it (Linguistic Motivation)

Because isiZulu is a highly agglutinative language, an individual author’s distinctive writing profile manifests as repeated usage of discrete morphemes, noun class prefixes, and verbal affixes rather than static standalone dictionary words.

1. The **1D Convolutional Layer** acts as a sliding morpho-syntactic feature extractor, scanning adjacent character/subword blocks to map word-building tendencies.
2. The **LSTM Layer** processes the chronological transition of these features across the paragraph, capturing sentence-level rhythms, structural pacing, and phrasing habits.
3. The **Global Skip Connection** mimics the lexical strength of traditional Bag-of-Words models, capturing fixed, document-wide topical vocabulary preferences.

### How it Does it (Layer Mechanics)

* **Input Layer:** Accepts sequence matrices of uniform shape `(None, 404)`.
* **Embedding Layer:** Projects discrete integer tokens into a continuous 128-dimensional dense space.
* **SpatialDropout1D (0.3):** Instead of dropping individual scalar elements inside vectors, it zeroes out entire 1-pixel subword tracks across the matrix. This eliminates sequence memorization and prevents the model from relying on localized phrases.
* **Conv1D Layer (128 Filters, Kernel=5):** Applies a sliding window of 5 subwords across the sequence with rectified linear activation (`ReLU`) to extract morphological features.
* **Batch Normalization:** Rescales and standardizes feature activations after the CNN, keeping the inputs to downstream layers uniform and accelerating gradient descent.
* **Track A (LSTM):** Downsamples spatial dimensions via `MaxPooling1D`, then feeds the sequence into a 64-unit `LSTM` to model sentence layout dependencies.
* **Track B (Global Skip):** Extracts the absolute strongest, document-wide stylistic activations via `GlobalMaxPooling1D`.
* **Concatenation Layer:** Fuses the 64-dimensional chronological feature vector with the 128-dimensional global feature map, creating a unified 192-dimensional stylometric signature.
* **Dense Head:** Passes the concatenated vector through a 64-unit fully connected layer, applies standard Batch Normalization, and projects to an 11-way `SoftMax` activation layer for output.

### Libraries Used

* `tensorflow.keras.layers`: To compile the non-linear network blueprint.
* `tensorflow.keras.models.Model`: For managing functional graph inputs and outputs.
